## 2. Luồng chức năng chính

- **Frontend SPA**: Điều hướng hash (#home, #products, #search, #cart, #profile, #contact, ...)
- **Backend API**: RESTful, trả JSON, bảo mật, phân quyền
- **Admin Panel**: Quản lý sản phẩm, user, đơn hàng, banner, dashboard
- **Database**: MySQL, chuẩn hóa, liên kết khóa ngoại
- **Uploads**: Lưu ảnh banner, sản phẩm, quản lý qua admin
- **Chat AI**: Nút nổi, proxy Gemini API, markdown, bảo mật key

## 3. Ghi chú
- Tất cả API endpoint đều nằm trong backend/api/
- Tất cả upload ảnh đều lưu trong uploads/ (không public qua git)
- File config_gemini.php không commit lên git
- Đảm bảo phân quyền rõ ràng giữa user và admin
- SPA chỉ có 1 file index.html, còn lại render động bằng JS
